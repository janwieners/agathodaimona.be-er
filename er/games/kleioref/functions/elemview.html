Die Elementarfunktion :view[]


Die Funktion :view[] erlaubt den Zugriff auf den Inhalt einer Eigenschaft, die einer Gruppe order einem Element mittels der Direktive property zugeordnet wurde.

Die Syntax erlaubt zwei Anwendungsweisen:

Die bloße Angabe der Funktion

:view[]

ohne weiteren Zusatz bewirkt, daß der Inhalt aller Property-Eigenschaften der aktuellen Gruppe ausgegeben wird.

Dagegen bewirkt

:view[:property]
daß nur der Inhalt der jeweiligen Property-Eigenschaft wiedergegeben wird.

So bewirkt z.B. die Angabe

:view[:authority]
daß innerhalb einer bestimmten Gruppe jeweils der Inhalt der Eigenschaft authority ausgegeben wird. Nimmt man das Beispiel, das im Zusammenhang mit der property-Direktive erläutert wurde, so bewirkt folgende Aufgabe:


query name=ceec;part=bibl
write part=:biblitem,:view[:authority]

unter anderem das folgende Ergebnis:


cec (1 = "kn28-0083ii") : biblitem     Hartzheim , p. 46-47, 53, 55, 139
cec (1 = "kn28-0083ii") : authority    Hollas: 150-151
cec (1 = "kn28-0083ii") : biblitem     Jaffe/Wattenbach, p. 29-31
cec (1 = "kn28-0083ii") : authority    Hollas: 150-151
cec (1 = "kn28-0083ii") : biblitem     C. H. Beeson, lsidor-Studien (Munich 1913), p. 70, 77, 90, 95, 102, 128
cec (1 = "kn28-0083ii") : authority    Hollas: 150-151
cec (1 = "kn28-0083ii") : biblitem     Bischoff, "Panorama der Handschriftendberlieferung...", Studien 3, p. 9 n.15 (English trans.
                                        "Manuscripts in the Age of Charlemagne", MLAC, p. 24)
cec (1 = "kn28-0083ii") : authority    Hollas: 150-151



Hier wird also die Information biblitem ausgegeben. Zusätzlich erfolgt jeweils die Ausgabe des Inhalts der Eigenschaft authority der des jeweiligen Elements.
