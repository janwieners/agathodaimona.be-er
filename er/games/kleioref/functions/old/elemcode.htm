Name: codebook

Input: Element or entry of data type text. Entries of other data types are converted.

Output: Set of entries of data type number or text.

Definition: Short description of the function, style of the old system manual

Argumentlist:

Argument No 1 (required)

This argument can accept different types of value

  • Alternative 1:Path with element as last component.

  • Alternative 2:Character string.

  • Alternative 3::query[]

Argument No 2 (optional)

This argument can accept different types of value

  • Alternative 1: Name of Thesaurus variable.
  • Alternative 4:Keyword:
    • /Number
    • /Text

  • Alternative 3:Path with element as last component.

  • Alternative 4:Keyword:
    • /Number
    • /Text

Argument No 3 (optional) Object name.
