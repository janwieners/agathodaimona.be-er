Name: catalogue

Input: Constant or element or entry.

Output: Set of groups.

Definition: Short description of the function, style of the old system manual

Argumentlist:

Argument No 1 (required) Object name.

Argument No 2 (optional) Access mode name.

Argument No 3 (optional)

This argument can accept different types of value.

  • Alternative 1: Character string.
  • Alternative 2: Search expression as character string.
  • Alternative 3: Path with element as last component.
