Die Funktion :dump[]

Die Elementarfunktion :dump[] dient dazu, den Inhalt lokaler Variablen auszugeben. Dump wird nach der Syntax :dump[variablenname] aufgerufen. Eine lokale Variable kann durch die Funktion :copy[] angelegt werden und z.B. den Inhalt eines Elements enthalten. Der Befehl

query name=dmr;part=:technik
write part=:copy[:technik,test],:order[],:dump[:test]
stop
würde also den Inhalt des Elements technik in die Variable test schreiben, dann die Ordnungsnummer des Dokuments und dann den Inhalt von test ausgeben. Das Resultat hat folgende Form:
Perlenstickerei
100
Perlenstickerei
Ebenso kann eine lokale Variable durch den Parameter current der convert- oder der condition-Direktive innerhalb einer Package-Vereinbarung definiert werden:
item name=karlsruhezoom;usage=package;overwrite=yes
bracket sensitive=yes
field start='http://gilgamesch2.hki.uni-koeln.de/promtest-cgi/kleioc/1010KlPTest/exec/zoom/%22';
    limit='%22/__kleioid/%22@<cgiuser@%22/__kleiopass/%22@<cgipass@%22';
convert current=zoomid;
    path=':convert[:dump[zoomid],zoomencode]'
exit name=karlsruhezoom

In diesem Beispiel wird der Inhalt des zu verpackenden Elements in der Variable zoom zwischengespeichert und mittels :dump[zoom] an geeigneter Stelle wieder ausgegeben.
