Der Parameter type

Der Parameter type bestimmt die Aktion, die ausgeführt werden soll, sobald ein Benutzerkonto abgelaufen ist bzw. in Begriff ist abzulaufen. Dies betrifft alle Konten, bei denen die definierte Zeitdistanz zwischen Verfallstermin und Bezugsdatum unterschritten wird.


Es gibt vier verschiedene Aktionen die beschreiben wie mit den betroffenen Benutzerkonten zu verfahren ist:

  1. warn: Erstellt eine Warnungsliste für den Systemadministrator.

  2. mail: Sendet eine Warnmail an die in der Benutzerverwaltung angegebene E-mailadresse.

  3. remove: Entfernt das abgelaufene Benutzerkonto und sendet eine verständigende email an den entsprechenden Benutzer

  4. kill: Entfernt den Benutzer ohne Verständigung

Die E-Mails werden unter zu Hilfename eines packages gebildet.
