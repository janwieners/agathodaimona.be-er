Der Parameter more:


Der Parameter erwartet die Schlüsselworte yes oder no. Letzteres ist Standard. Der Parameter bewirkt, daß Mehrfacheinträge innerhalb von Elementen jeweils aus einer Zeile ausgegeben werden.

Folgende Aufgabe:

query name=basdat;part=per
write part=:each[];more=yes
stop


erzeugt daher diese Ausgabe:

per (2 = "DOK050295")
          idd          DOK050295
          name         ABELE  
          vorname      JOHANN ; FRIEDRICH
          geschlecht   maennlich
          hko          OTTMARSHEIM
          land         OA MARBACH


Der Vorname " JOHANN ; FRIEDRICH" erscheint auf einer Zeile.
