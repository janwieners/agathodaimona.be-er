Der Parameter image:

Der Parameter image steuert die Ausgabe von Daten des Type image. Der Standardwert, der dem Schlüsselwort no entspricht, weist das System an, Bilder über das entsprechende Ausgabegerät (z.B. den Bildschirm) auszugeben. Der Parameterwert postscript veranlaßt die Ausgabe in einer Postscriptdatei, die standardmäßig den Name der Bilddatei und mit der Erweiterung "ps" trägt  Der Parameter interactive bewirkt die Ausgabe in GUI-Umwelten (graphic user interface).
