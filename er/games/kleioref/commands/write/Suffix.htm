Der  Parameter suffix:

Dieser Parameter erwartet eine Zeichenkette, die in Hochkommata angegeben werden. Daten des Typs image können  als Postscriptdateien ausgegeben werden (vgl. hierzu den Parameter image ). Dabei gilt der Name der Bilddatei mit der Erweiterung "ps" als Name der Postscriptdatei. Durch den Parameter suffix kann eine frei wählbare andere Erweiterung angegeben werden.
