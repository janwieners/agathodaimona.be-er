Der  Parameter sequence


Als Parameterwert erwartet sequence eine ganze Zahl. Die Zahl entspricht der Anzahl  der Ausgabeeinheiten, nach denen das System einen Seitenvorschub ausgibt. In Verbindung mit dem Parameter comment läßt sich sequence dazu verwenden, um nach einer bestimmen Zahl von Ausgaben eine Konstante auszugeben.

Der Befehl

query name=basdat;part=per
write part=:name;comment="Listing Nr. 10";
                         sequence=20;
stop


bewirkt also, daß nach 20 Zeilen die Zeichenkette "Listing Nr. 10" ausgegeben wird:
