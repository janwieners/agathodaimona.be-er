Der Parameter comment:


Dieser Parameter erwartet einen Zeichenkette als Konstante, die in Hochkommata angegeben werden muß. Der Inhalt der Konstante wird standardmäßig nach jeder Ausgabeeinheit ausgegeben:

Folgende Aufgabe

write part=:name;comment="Listing Nr. 10"

liefert daher folgendes Ergebnis:

per (1 = "DOK030011") : name         ABELE  
Listing Nr. 10
per (2 = "DOK050295") : name         ABELE  
Listing Nr. 10
per (3 = "DOK001191") : name         ABT    
Listing Nr. 10
per (4 = "DOK000085") : name         ABT    
Listing Nr. 10
per (5 = "DOK001199") : name         ACKER  
Listing Nr. 10


Sinnvoll kann der Parameter im Verbindung mit dem Parameter sequence eingesetzt werden, der die Anzahl der Ausgabeeinheiten festlegt, nach denen der Inhalt von comment ausgegeben wird.
