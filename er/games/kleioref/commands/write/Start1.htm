Der Parameter start:

Der Parameter start erwartet als Parameterwert die Schlüsselworte yes oder no. Ersteres ist das Standardverhalten. Die Angabe start=no bewirkt, daß die Namen der Informationsgruppen bei der Datenausgabe unterdrückt werden.


Beispiel:

write=:name;start=no

Dieser Parameter wird sinnvollerweise häufig zusammen mit dem Parameter position =no angegeben.
