Der Parameter self:

Der Parameter erwartet die Schlüsselworte yes oder no. Ersteres ist Standard. Die Angabe self=no bewirkt, daß die Bezeichnungen der Elemente nicht ausgegeben werden.


Die Aufgabe

query name=basdat;part=per
write part=:each[];self=no
stop 
     
liefert folgendes Ergebnis:
