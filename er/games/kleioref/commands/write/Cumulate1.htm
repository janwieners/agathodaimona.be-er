Der Parameter cumulate:


Der write-Befehl bewirkt normalerweise, daß der Inhalt jedes Elements auf einer neuen Zeile ausgegeben wird. Durch die Einstellung cumulate=yes (no ist Standard) kann das Systemverhalten dahingehend verändert werden, daß so viele Elemente wie möglich auf einer Zeile ausgegeben werden.

Die Aufgabe

query name=basdat;part=per
write part=:each[];cumulate=yes
stop 
     

erzeugt damit folgendes Ergebnis:

per (1 = "DOK030011") idd = DOK030011 / name = ABELE / vorname = CARL / geschlecht = maennlich / hko = KIRCHHEIM,TECK / land = OA KIRCHHEIM
