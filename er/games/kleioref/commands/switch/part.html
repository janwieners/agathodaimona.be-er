Der Parameter part

Der Parameter part erwartet als Parameterwert eine Pfadangabe bzw. ein einzelnes Element. Er bezeichnet dasjenige Element, dessen Inhalt als Auswahl riterium für das "switching"  der Ausführungsblöcke herangezogen wird. Typischerweise wird man den Parameter in folgender Form einsetzen: 

query name=kleio;part=/catalogue[kuenstler,complete,"picasso"]
switch part=:database[];name=selectDB
...
exit name=selectDB


Das heißt also, daß nachfolgend die Auswahl der bedingten Kommandoblöcke sich am Wert orientiert, den :database[] zurückliefert. Das Vergleichsmuster, das im part-Parameter der block-Direktive  angegeben wird bestimmt demnach die Ausführung eines Kommandos innerhalb eines Blocks.
