Der Parameter name

Der Parameter name erwartet als Parameterwert eine frei wählbare Zeichenkette. Der Parameter legt einen Namen fest, unter dem der Ablaufrahmen des switch-Kommandos bekannt ist. Derselbe Name wird wiederum im name-Parameter der der exit-Parameter beim Abschluß der Umwelt angegeben. Die Angabe


switch part=:database[];name=selectDB

....
exit name=selectDB

definiert also eine Kommandoumwelt des Namens "selectDB", innerhalb derer eine beliebige Anzahl einzelner Kommandoblöcke eingerichtet werden können.
