Der Parameter target:

Der Paremeter target erwartet als Parameterwert die Angabe eines Verzeichnisses, das als Konstante (also in Hochkommata) sepzifiziert werden muß. Die Notation hängt vom jeweiligen Betriebssystem ab.

Bestimmte Definitionen, die durch den item-Befehl angelegt werden, erzeugen eigene eigene physikalische Dateien (z.B. codebooks). Der Parameter target gibt an, wo diese Dateien abgelegt werden sollen. Als Default-Wert gilt das aktuelle Verzeichnis, in der der item-Befehl ausgeführt wird.
