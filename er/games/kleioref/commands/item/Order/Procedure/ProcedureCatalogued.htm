Die Direktive catalogue

Die Direktive catalogue erfüllt im Rahmen der Definition einer Prozedur eine Schlüsselfunktion: 

Sie gibt die Quelle an, unter welcher das zugehörige Skript zu finden ist. Dabei kann das Skript grundsätzlich direkt im source-Parameter eingetragen werden oder aber auch in einer gesonderten Programm- oder Quelldatei liegen, die dann entsprechend benannt werden muß.
