Der Parameter source

Der Parameter source gibt an, aus welcher Quelle der Quelltext der Prozedur gelesen werden soll. Der Wert von source richtet sich nach dem Parameterwert der Direktive type.

Wir bei type der Wert text (Standardannahme) spezifiziert, so kann der Quelltext der Prozedur direkt im source-Parameter eingetragen werden. Zeilenumbrüche innerhalb des Quelltextes müssen dabei mit "\n" angegeben werden.

Die folgende Aufgabe entspricht also einer vollgültigen Definition einer katalogisierten Prozedur (der Parameter type könnte weggelassen werden, da der Wert "text" als Standard gesetzt ist).

item name=job1;usage=procedure;overwirte=yes
catalogue source="query name=arachne;part=row\nwrite part=:each[]";type=text
exit name=job1

Es wird also hier die Aufgabe

query name=arachne;part=row
write part=:each[]

unter dem Namen "job1" katalogisiert!

Daneben können Dateien als Quellen angegeben werden. Der Dateityp wird durch den Parameter type bestimmt.
