Der Parameter type

Der Parameter type erwartet die Parameterwerte

  • text
  • file
  • programme
Der Wert text ordnet an, daß der Inhalt der katalogisierten Prozedur als Konstante unmittelbar im source-Parameter eingetragen wird. Dieser Wer ist das Standardverhalten des Systems.
Der Wert file sieht vor, daß die jeweilige Aufgabe in einer Datei abgelegt ist, die im ASCII-Format vorliegen muß. Es sind hier alle Dateinamen und -endungen erlaubt, die das jeweilige Betriebssystem zuläßt.
Der Wert programme bezieht sich auf Dateien, die durch das GUI angelegt wurden und als Programmdateien die Endung "k~p" tragen. Bis zum weiteren Ausbau des GUI ist dieser Parameter eher Ankündigung zu verstehen.
