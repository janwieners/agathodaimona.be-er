Die Direktive project

Über die Project Direktive können Projekte des Users festgelegt werden. Es gibt die Parameter

  • name und

  • overwrite



Der Parameter name

Der Parameter name erwartet als Parameterwert eine Zeichenkette, die in Hochkommata angegeben werden muß.
Es ist kein Standartwert voreingetellt, gängig ist jedoch hier den usernamen als default einzusetzen.

Der Parameter Overwrite

Der Parameter overwrite kann die Werte yes oder no annehmen. Letzteres ist Standard und ordnet an, dass Projekte nicht überschrieben werden können.
