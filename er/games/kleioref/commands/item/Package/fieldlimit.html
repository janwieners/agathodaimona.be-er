fieldlimit

Der Parameter limit:

Der Parameter limit erwartet als Parameterwert eine Konstante, die folglich in Hochkommata angegeben werden muß. Wenn die betreffende Package-Vereinbarung durch die Funktion :package[] oder den Parameter package aufgerufen wird, so wird der Inhalt des limit-Parameters nach der zu verpackenden Information ausgegeben.

Ein Beispiel verdeutlicht diese Funktion. Man habe folgende minimale Package-Definition:

item name=verpackung;usage=package
fields limit="Endtag "
exit name=verpackung

In diesem Fall würde folgende simple Aufgabe

query name=kurs;part=m
index part=:package[:name,verpackung]

zu folgendem Ergebnis führen:

Abt Endtag FRE-56-567
Agner Endtag FRE-55-1
Arnold Endtag FRE-57-35
    ...
Zillinger Endtag FRE-59-297
Zoller Endtag FRE-59-306
Zorn Endtag FRE-58-880

Man kann den Parameter limit auch als Komplementärparameter von start betrachten. So wie start den Beginn des zu verpackenden Inhalts auszeichnet, kann mit limit dessen Ende ausgezeichnet werden. Ein realistisches Package, das z.B. unter HTML Fettdruck einstellen würde, hätte demnach folgende Form:

item name=fett;usage=package
fields start="<b>";limit="</b>"
exit name=fett
