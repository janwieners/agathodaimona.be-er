Der Parameter start

Der Parameter start erwartet als Parameterwert eine Zeichenkette, die als Konstante und folglich in Hochkommata angegeben werden muß.

Der Wert entspricht dem Markupbestandteil, um den der zu verpackende Inhalt am Anfang ergänzt wird, um ggf. ein ausgewogenes Markup herstellen zu könnnen. Der Parameter wird also in folgender Form eingesetzt:

bracket start="\n<tr>";
        limit="</tr>\n<tr>\n"
