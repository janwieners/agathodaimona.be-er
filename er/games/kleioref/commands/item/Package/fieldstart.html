fieldstart

Der Parameter start:

Der Parameter start erwartet als Parameterwert eine Konstante, die folglich in Hochkommata angegeben werden muß. Wenn die betreffende Package-Vereinbarung durch die Funktion :package[] oder den Parameter package aufgerufen wird, so wird der Inhalt des start-Parameters vor der zu verpackenden Information ausgegeben.

Ein Beispiel verdeutlicht diese Funktion. Man habe folgende minimale Package-Definition:

item name=verpackung;usage=package
fields start="Starttag "
exit name=verpackung

In diesem Fall würde folgende simple Aufgabe

query name=kurs;part=m
index part=:package[:name,verpackung]

zu folgendem Ergebnis führen:

Starttag Abt FRE-56-567
Starttag Agner FRE-55-1
Starttag Arnold FRE-57-35
    ...
Starttag Zillinger FRE-59-297
Starttag Zoller FRE-59-306
Starttag Zorn FRE-58-880

Man kann den Parameter start auch als Komplementärparameter von limit betrachten. So wie start den Beginn des zu verpackenden Inhalts auszeichnet, kann mit limit dessen Ende ausgezeichnet werden. Ein realistisches Package, das z.B. unter HTML Fettdruck einstellen würde, hätte demnach folgende Form:

item name=fett;usage=package
fields start="<b>";limit="</b>"
exit name=fett
