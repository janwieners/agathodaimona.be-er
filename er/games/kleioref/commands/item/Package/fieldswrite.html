fieldwrite

Der Parameter write:

Der Parameter write erwartet als Parameterwert eine Konstante, die folglich in Hochkommata angegeben werden muß. Wenn die betreffende Package-Vereinbarung aufgerufen wird, so wird der Inhalt des write-Parameters vor dem Namen des zu verpackenden Elements ausgegeben.

Ein Beispiel verdeutlicht diese Funktion. Man habe folgende minimale Package-Definition:

item name=verpackung;usage=package;overwrite=yes
fields  write="<h3>"
exit name=verpackung

Unter Verwendung folgenden einfachen write-Befehls

query name=kurs;part=hei
write part=k:each[=noheaders,=packaged,:vorname,verpackung]

erzeugt dieser Parameter folgende Ausgab:e


<h3>vornameFriedrich
          Gottlob

<h3>vornameCarl
          Christian

<h3>vornameMarie
          Barbara

<h3>vornameJohann
          Friedrich

Der Parameter write ist Komplementärparameter von connect. So wie write den Beginn des zu verpackenden Namens auszeichnet, kann mit connect dessen Ende ausgezeichnet werden. Ein realistisches Package, das z.B. unter HTML den Namen des Elements als Überschrift definieren würde, hätte demnach folgende Form:

item name=head;usage=package;overwrite=yes
fields  write="<h3>";
   connect="</h3>";
exit name=head
