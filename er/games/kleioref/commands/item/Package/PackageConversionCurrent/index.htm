Der Parameter current

Der Parameter current erwartet als Parameter den frei wählbaren Namen einer Variable. Der Parameterwert wird damit zu einer "Stand-in-Variable", die adhoc in das Package eingeführt wird und durch die Direktive conversion weiter verarbeitet wird. Es handelt sich dabei um den Inhalt des Elements, das in der fields-Direktive verpackt wird.

Eine typische Anwendung hat die folgende Form:

field start='Starttag';
    limit='Endtag'
convert
    current=dbimageid;
    path=':package[:dump[dbimageid],uustbzoom]'

Hier wird also der Inhalt des Elements das in der Fields-Direktive verpackt wird, wiederum in die Stand-in-Variable "dbimageid" geschrieben, die dann ihrerseits im Parameter path mit :dump[] aufgerufen und  nach den Regeln des Package "uustbzoom" verpackt wird.
