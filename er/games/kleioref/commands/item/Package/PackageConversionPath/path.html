convert - path

Der Parameter path

Der Parameter path erwartet als Parameterwert eine Zeichenkette als Konstante. Der Parameterwert wird dementsprechend in einfachen oder doppelten Hochkommata angegeben.
Der Wert des Parameters path gibt den Pfad an, in den die durch die Direktive convert behandelte Information eingestellt werden soll. Dabei ist "Pfad" hier nicht restriktiv im ausschließlichen Sinne des Dateisystems auf einem Rechner zu verstehen. "Pfad" meint hier auch jede syntaktisch legale Veränderung der Daten durch Zeichenkettenoperationen, Einbaufunktionen u.ä., wie das folgende Beispiel zeigt.

convert current=thumbnailid;
     path=':form["http://gilgamesch2.hki.uni-koeln.de/Prometheus1/dadaweb/thumbs/T"]
    |+:convert[:dump[thumbnailid],dos2unix]|+:form[".jpg"]'


Hier wird also der wert einer Variablen (konkret der Name einer Bilddatei) um die Pfadangabe "http://gilgamesch2.hki.uni-koeln.de/Prometheus1/dadaweb/thumbs/T" sowie um die Endung ".jpg" ergänzt. Zugleich wird der Wert der Variable durch die Conversionsvereinbarung "dos2unix" behandelt.
