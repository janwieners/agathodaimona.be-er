Der Parameter Name:

Der Parameter name erwartet als Parameterwert eine frei wählbare Zeichenkette. Es wird hier eine Name festgelegt unter dem das betreffende logische Objekt später angesprochen wird.

Ein Beipsiel hierfür ist folgendes Bestandteil eines item-Befehls:



item name=definition



Hier wird also ein logisches Objekt in die Umwelt eingeführt, das später unter dem Namen „definition“ angesprochen werden kann.
