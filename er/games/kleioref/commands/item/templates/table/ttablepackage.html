Der Parameter package

Der Parameter package erwartet als Wert den Namen eines logischen Objekt des Klasse Package. Dieses Package legt fest, wie die aus einem Container abgeleiteten Informationen zu einem Bild in eine Tabelle geschrieben werden sollen.

Das folgende Package definiert eine entsprechende Tabelle.

item name=desmap-i0;usage=package;overwrite=yes
fields start='<td height="90" width="140" align=center bgcolor="#CCCCCC">';      
        first='</td><td height="90" width="10" valign=bottom bgcolor="#CCCCCC">';
       connect='<td height="90" width="10"></td>
        <td height="90" width="140" align=center bgcolor="#CCCCCC">';
       limit='<td height="90" width="10"></td>';
       current=tabcoord;
       suspend=nextbatch
               
convert current=folderbaseimageid;
        path=':package[:dump[folderbaseimageid],folderimage]';
convert usage=first;
        path=':package[:dump[tabcoord]&+:package[:dump[folderbaseimageid],folderref],folderpopup]';
exit name=desmap-i0


In diesem Package passiert - in groben Zügen - folgendes:
  • Es werden für jede Bildnummer im Container je drei Tabellenzellen angelegt. Die erste Zelle enthält das Bild als Thumbnail. Die Umsetzung der Bildnummer zu einem entsprechenden Bild-URL erfolgt in der ersten convert-Direktive des Package durch das conditional Package folderimage. In die zweiten Zelle wird ein Pfeil eingefügt, der als graphischer Link mit verschiedenen Funktionen versehen wird. Dafür ist die zweite convert-Direktive ("usage=first") verantwortlich. Die dritte Zelle bleibt leer und dient als Abstandhalter zu nächsten Zelle (hier zur Kenntlichmachung rot gefäbt. Durch das Package realiter aber ohne Hintergrundfarbe).

Der entsprechende Aufruf des oben angeführten package erfolgt im Parameter package.
