Der Parameter self

Der Parameter self erwartet als Wert den Namen eines logischen Objekt des Klasse Package. Dieses Package wird dann ausgeführt, wenn innerhalb einer Tabelle, die durch den Parameter package definiert wird, einzelne Tabellenzellen leer bleiben, weil es keinen zu verpackenden Inhalt gibt. Dies trifft dann zu, wenn die Anzahl der Bilder, die auf der letzten seite einer Arbeitsmappe dargestellt werden soll, geringer ist als die Anzahl der reservierten Bildzellen. Es handelt sich also um eine typische Default-Einstellung.

Mann kann dann als Defaultwert z.B. einstellen, daß anstelle des Bildes eine bestimmte Graphikdatei ausgegeben werden soll. Ein solches Package kann die folgende Form haben. 


item name=emptypraes;usage=package;overwrite=yes
fields start='<a href="#" onClick="menuRaiser(';
       limit=',\'\');"><img src="http://prometheus.hki.uni-koeln.de/Prometheus1/abb/p-l.jpg" border=0></a>';
exit name=emptypraes

Es wird also in den Fall, in dem kein Bild innerhalb der Tabelle verpackt werden kann, die im URL angegebene Graphik "p-l.jpg"  verpackt. Damit wird gewährleistet, daß die Tabellenzelle "ordentlich" gefüllt ist und die Tabelle korrekt dargestellt wird. Syntaktisch hat die entsprechende Direktive die folgende Form:

table package=desmap-i0;self=emptymap

Das also heißt, daß die Tabelle nach den Regeln des Pakets "desmap-i0" erstellt wird, während fehlende Bilder durch den Defaultwert in "emptymap" ersetzt werden.
