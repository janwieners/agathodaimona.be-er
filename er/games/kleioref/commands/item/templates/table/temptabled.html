Die table-Direktive

Der Parameter package in der frame-Direktive des Objekts template ruft eine Package auf, das allgemein eine HTML-Seite definiert.

Dagegen bestimmt die Direktive table das Aussehen einer Tabelle, in der die einzelnen Objekte des Conatiners dargestellt werden. Die Darstellung kann dabei - benutzerfeiniert - als Bildansicht, Bild/Textansicht oder nur Textansicht erfolgen. Die Tabelle wiederum wird in die Seite integriert, die durch die frame-Direktive definiert ist.

In jedem Falle müssen die jeweiligen Informationen tabellarisch formatiert werden.
