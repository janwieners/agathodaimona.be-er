Der Parameter sensitive

Der Parameter sensitive erwartet als Parameterwert die Schlüsselworte yes oder no, die auf den ersten Buchstaben abgekürzt werden können. Der Standard in no. Der Wert yes ordnet an, daß die Container in usersensitiv abgelegt werden. Das heißt, daß für die einzelnen User entsprechende Unterverzeichnisbäume abhängig vom Stammverzeichnis (vgl. Parameter target) angelegt werden.

Eine entsprechende Direktive hat folgende Form:

frame package=desmap-a1;target="/hdg/Prometheus2/mappen";sensitive=yes
