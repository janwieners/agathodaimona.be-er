Der Parameter project

Der Parameter project erwartet als Parameterwert die Schlüsselworte yes oder no, die auf den ersten Buchstaben abgekürzt werden können. Der Standard in no. Der Wert yes ordnet an, daß ein User mehrere Projekte verwalten kann. Realiter heißt dies, daß dem Verzeichnis eines Users wiederum beliebig viele Unterverzeichnisse für jeweils beliebig viele  Container zugeordnet werden können.

Eine entsprechende Direktive hat folgende Form:

frame package=desmap-a1;target="/hdg/Prometheus2/mappen";project=yes; sensitive=yes
