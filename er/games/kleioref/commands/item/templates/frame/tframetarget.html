Der Parameter target

Der Parameter target erwartet als Parameterwert die Angabe eine Pfades, der als Konstante, also in Hochkommata angegeben werden muß. Das hier angegebene Dateiverzeichnis entspricht dem Wurzelverzeichnis, in dem die container (=Arbeitsmappen bzw. Präsentationen) abgelegt werden. Für jede Usergruppe und abhängig von dieser wiederum für jeden User werden davon ausgehend weitere Verzeichnisse angelegt. Die Zeile

frame target="/hdg/Prometheus2/mappen";

würde anordnen, daß die Container in einer Verzeichnisstruktur abgelegt werden, die vom Verzeichnis "/hdg/Prometheus2/mappen" ihren Ausgang nimmt.
