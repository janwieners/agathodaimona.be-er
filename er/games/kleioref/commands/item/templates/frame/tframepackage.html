Der Parameter package

Der Parameter package erwartet als Parameterwert den Namen einer  logischen Vereinbarung der Klasse package. Das Package, das hier angegeben wird, bestimmt den grundsätzlichen Aufbau einer Arbeitsmappe oder Präsentation in Gestalt einer Webpage. Es wird also gleichsam das Seitengerüst erzeugt. Dazu muß das template folgende Grundstruktur ausweisen:

item name=desmap0;usage=template;
          frame package=desmap-a1

...

exit name=desmap0

Es wird also im template mit dem Namen "desmap0" auf ein Package des Namens desmap-a1 verwiesen, das seinerseits folgende Struktur haben könnte:

item name=desmap-a1;usage=package;overwrite=yes
brackets sensitive=yes
fields origin='<html>\n
        <head>\n
                <title>AM / Version 1 v. 2.3.2002</title>\n
        </head>\n
        <body bgcolor="#999999">\n
        <table border=0 cellspacing="0" width="100%">';
   complete='</table>'
exit name=desmap-a1

Damit ist also eine Seite definiert, in die als Content eine Arbeitsmappe oder Präsentation eingeladen werden können.
