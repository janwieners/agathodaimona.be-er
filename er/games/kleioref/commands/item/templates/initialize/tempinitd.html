Die initialize-Direktive

Die Direktive initialize setzt voraus, daß in der frame-Direktive der Parameter project mit dem Wert yes spezifiziert wurde. Wenn dies zutrifft, so unterscheidet das System zwischen verschiedenen Projekten. Das heißt, es werden unterhalb des Wurzelverzeichnisses verschiedene Directorys angelegt, die ihrerseits mehrere Container enthalten können.

Jedes dieser Verzeichnisse entspricht einem "Projekt". Aufgabe der Direktive initialize ist es, einen Defaultwert für das erste Verzeichnis und die drin enthaltene erste Datei festzulegen.

Die Angabe

initialize name=mappe;project=projekt


legt also fest, daß unterhalb des Userverzeichnisses ein Verzeichnis "projekt" angelegt wird, worin der erste Container den Namen mappe tragen soll.
