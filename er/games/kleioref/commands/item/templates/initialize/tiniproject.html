Der Parameter project

Der Parameter project erwartet als Name eine frei wählbare Zeichenkette. Diese entspricht dem Defaultwert des Verzeichnisses in dem die Container abgelegt werden.

initialize project=projekt

legt also fest, daß die Container standardmäßig im Verzeichnis "projekt" abgelegt werden, das seinerseits unterhalb des Stammverzeichnisses liegt, das im target-Parameter der frame-Direktive spezifiziert wurde.
