Die path-Direktive

Die Direktive path dient zur Anforderung von Textinformationen, die innerhalb eines templates verpackt werden sollen. Die path-Direktive kann/muß, wenn die Informationen aus verschiedenen Datenbanken stammen, mehrfach angegeben werden. Die path-Direktiven beziehen sich dann immer auf die Variable, die zuvor mittels der Direktive variable eingeführt worden sein muß. Diese Zuweisung an eine bestimmte Variable ist so lange gültig, bis eine neue Variable-Direktive gesetzt wird, auf die sich dann wiederum die nachfolgenden path-Direktiven beziehen.
