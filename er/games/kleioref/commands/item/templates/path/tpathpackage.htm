Der Paramater Package

Der Parameter package erwartet als Parameterwert den Namen einer Vereinbarung der Klasse package. Die darin angegebenen Verpackungsanweisungen werden auf den Inhalt angewendet, der durch die jeweilige path-Direktive ausgewählt ist. Das Package bestimmt die optische Wiedergabe der textuellen Information. Üblicherweise wird man hier z.B. eine Tabellenzelle o.ä. generieren. Es ist aber auch z.B. möglich, den Inhalt mit anderen Druckkonstanten zu versehen. Man habe folgendes Package:

item name=desmap1-i1;usage=package;overwrite=yes
fields start='<td valign=top height="20" width="14%">';limit='</td>\n';
exit name=desmap1-i1

Es wird hier also im Prinzip eine Tabellenzelle gesetzt in der die Information eingefügt wird. Entsprechend hat die path-Direktive folgende Form:

path source=archgiessen;part=':p_künstler';self="&nbsp;";package=desmap1-i1
