Der Parmater self

Der Parameter self erwartet als Parameterwert eine Konstante, die also in Hochkommata angegeben werden muß!

Der Parameter dient zur Einstellung eines Defaultwerts, wenn

Die path-Direktive hat damit folgende Form:

path source=uustb;self="keine Angabe";package=desmap1-i1;part="/root[0]/aob30:a3100";

Hier wird also der Wer "keine Angabe" ausgebene, sobald das Element "a3100" in einem bestimmten Dokument keinen Inhalt aufweist.
