Die variable-Direktive

Die variable-Direktive erlaubt innerhalb einer template-Umgebung die Anlage von Variablen, in denen  Textinformationen gesammelt werden können, die dann zusätzlich oder anstelle der zu verpackenden Bilder ausgegeben werden.

Dem Mechanismus liegt folgendes Prinzip zugrunde: Jede Bildnummern im Container wird bei Aufruf des Templates im Katalog der Bildnummern indentifiziert, der in der frame-Direktive spezifiziert werden muß. Damit wiederum kann das Ausgangsdokument jeder Bildnummer in der entsprechenden Quelldatenbank bestimmt werden. Damit wiederum ist es möglich die jeweiligen Metadaten zum Bild abzurufen.

Die Metadaten der einzelnen Bilder können dann in Variablen gesammelt und ausgegeben werden. Dazu müssen ein symbolischer Name und der Inhalt der entsprechenden Variable bestimmt werden. Die Vergabe des Namens erfolgt durch den Parameter name der Direkte variable. Der Inhalt der Variable wird dann durch nachfolgende path-Direktiven bestimmt. Die Konstruktion hat damit folgende Form:

variable name=des-name
path part=':form["&nbsp;"]';
     self="&nbsp;";package=desmap1-i1
path source=archgiessen;part=':p_künstler';
     self="&nbsp;";package=desmap1-i1
path source=wbarchiv;part=':künstler';
     self="&nbsp;";package=desmap1-i1


Es wird also eine Variable mit dem Namen "des-name" angelegt, deren Inhalt durch die nachfolgenden drei path-Direktiven definiert ist.
