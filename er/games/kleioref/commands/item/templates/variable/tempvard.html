Die variable-Direktive

Die Direktive variable erlaubt die Defitinition von Variablen, in denen der Inhalt verschiedener Datenbankeinträge gesammelt werden kann. Dabei können die entsprechenden Informationen in einer oder mehreren getrennten Datenbanken vorliegen.

Die Direktive kennt gegenwärtig nur den Parameter name, der der Variablen einen Name zuweist

Alle path-Dirketiven, die auf eine variable-Direktive folgen, sind relativ zu dieser. das heißt, sie "füllen" die Variable mit Inhalt. Dies gilt so lange, bis eine neue variable-Direktive eingeführt wird.
