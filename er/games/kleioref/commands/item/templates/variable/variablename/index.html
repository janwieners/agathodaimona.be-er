Der Parameter name

Der Parameter name erwartet als Wert Variablennamen, der den entsprechenden Regeln folgen muß.

Die Direktive

variable name=des-titel
path part=':form["&nbsp;"]';
self="&nbsp;";package=desmap1-i1
path source=archgiessen;part=":p_titel";
self="&nbsp;";package=desmap1-i1

weist also der temporären Variable den Namen "des-titel" zu. Die nachfolgenden path-Direktiven füllen die Variable mit Inhalt.
