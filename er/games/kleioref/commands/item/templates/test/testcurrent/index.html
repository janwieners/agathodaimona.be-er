Der Parameter current

Der Parameter current erwartet als Parameterwert einen frei wählbaren Namen einer lokalen Variable. Die jeweils getestete Zeichenkette aus dem Container wird unter der besagten Variable abgelegt und steht dann für komplexere Operationen temporär zur Verfügung. D.h., sie kann mittels der Funktion :dump[] lokal ausgegeben und bearbeitet werden.

Dabei können sämtliche in Kleio zulässigen Zeichenkettenoperationen ausgeführt werden. Z.B.:

test current=variable;
part=':substr[:conv[:dump[:argument],imagounify],:size[:conv[:dump[variable],imagounify]]&-:form["15",number],16]'
