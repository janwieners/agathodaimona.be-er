Der Parameter name

Der Parameter name erwartet als Parameterwert den Namen einer benannten  Ersetzungsvariablen (Ersetzungsmarker), wie z.B. @=person@.. Es ist dies dann derjenige Ersetzungsmarker, der durch die Direktive initial einen Startwert zugewiesen bekommt. Wenn innerhalb einer katalogisierten Prozedur mehrere benannte Ersetzungsmarker vorkommen, muß die Direktive initial (und damit der Parameter name) entsprechend oft wiederholt werden:

item name=search;usage=procedure;overwrite=yes
catalogue source="search.txt";type=file
initial name=cat2use;text="kuenstler"
initial name=term;text="a*"
initial name=dir;text="after"
exit name=search

Es werden also hier nacheinander Startwerte für je je Ersetzungsvariable definiert, die jeweils durch eine neue initial-Direktive und einen neuen name-Parameter angesprochen werden.
