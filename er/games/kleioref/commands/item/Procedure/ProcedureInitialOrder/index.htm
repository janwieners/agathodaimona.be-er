Der Parameter order

Der Parameter order benennt einen  gezählten Ersetzungsmarker, wie z.B. @#3@.  Als Parameterwert erwartet order dementsprechend eine ganze Zahl. 

Die Direktive

initial order=3;text="picasso"

weist also dem Ersetzungsmarker @#3@ den Startwert "picasso" zu. Wenn mehrere gezählt Ersetzungsmarker in einer Aufgabe vorkommen, muß die Direktive initial mit einem jeweils anderen Wert des order-Parameter entsprechend oft wiederholt werden.
