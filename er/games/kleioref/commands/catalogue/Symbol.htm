Der Parameter symbol


Der Parameter symbol erwartet als Parameterwert ein einzelnes, frei wählbares Zeichen. Dieses Zeichen dient bei der Abfrage des Katalogs als Trunkierungssymbol. Das heißt, es kann als Wildcard eingesetzt werden, um "alles ab einschließlich einer bestimmten Zeichenkette" als Vergleichsmuster im Katalog zu suchen. 

Man kann also mit dem Parameter symbol nach folgender Syntax einen Katalog erzeugen:

query name=datenbank;part=:kuenstlername
catalogue name=kuenstler;part=:name;overwrite=yes;
          symbol=*
     

Damit ist es nun möglich unter Einsatz des Trunkierungssymbols "*" nach allen Künstlern zu suchen, deren Eintrag mit einer bestimmten Zeichenfolge beginnt. Die folgende Aufgabe

query name=kleio;part=/catalogue[kuenstler,complete,"RUB*"]:order[]=not null
write

liefert daher folgendes Ergebnis:

row (301 = "33290")
          vorname      Peter Paul
          recordid     33290
          nachname     Rubens
          datierung_1  1612
          ort          Vaduz
          aufbewahrungsort Galerie Lichtenstein
          titel_uebersetzung Die Toilette der Venus
          abbildungsnachweis Gerstenberg: Velazquez, S. 231, Abb 193
          schlagwörter Venus
          gespeichert_unter e:\bilder\231199\bild14a
          medium       Malerei
          eingabedatum 23.11.1999
          gdest        231199bild14a


Es wurde als der Künstler "Rubens" gefunden, obwohl der Katalog "nur" nach "RUB" durchsucht worden war.
