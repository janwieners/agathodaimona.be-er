Der Parameter source

Der Parameter source erwartet als Parameterwert den Namen einer Datenbank, die innerhalb eines database-Kommandos definiert sein muß. Es können hier als kommaseparierte Liste auch die Namen mehrerer Datenbanken angegeben werden. 

Die Angabe bewirkt, daß die entsprechende aktuelle Umwelt sich ausschließlich auf die gelisteten Datenbanken bezieht. Dies bedeutet auch, daß z.B.  Kataloge, die sich auf mehrere Datenbanken beziehen, ausschließlich den Datenbestand indexieren, der in denjenigen Datenbanken enthalten ist, die in der aktuellen Umwelt zugelassen sind.

Im folgenden Beispiel existieren in der Umwelt eine ganze Reihe von Datenbanken und ein Katalog "institution", der sich auf alle Datenbanken bezieht. Für eine bestimmte Auswertung soll aber in der Umwelt nur die Datenbank "archgiessen" existieren. Es wird also im source-Parameter des environment-Befehls der entsprechende Wert spezifiziert. Wie der nachfolgende Index-Befehl zeigt, werden tatsächlich nur die EinTräge des Katalogs herangezogen, die aus der Datenbank "archgiessen" herrühren.


environment source=archgiessen
query name=kleio;part=/catalogue[institution,complete,"a*"]
index part=:database[];maxi=1
cont
