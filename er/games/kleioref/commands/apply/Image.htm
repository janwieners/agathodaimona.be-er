Der Parameter image

Der Parameter image ist obligatorisch. Er erwartet als Parameterwert die den Namen eines logischen Objekts der klassische Image, das den Ablaufrahmen eines apply-Kommandos festlegt.

Typischerweise hat ein solches Objekt die folgende Form:

item name=raw;usage=image
part text=yes
device name=scans;target=".";total=yes
exit name=raw

Hier wird in der Dirtektive device festgelegt, daß die Quelldaten (also die umzuwandelnden Bilddaten) sich im Verzeichnis "scans" befinden, das seinerseits vom aktuellen Verzeichnis abhängig ist (Parameter trget der Direktive device).

Der apply-Befehl nimmt auf diese Voreinstellungen Bezug, indem der Parameter image den Wert "raw" annimmt, der als frei gewählter Name für das Onjekt des Typs image vergeben wurde:

apply image=raw;
    target="res800x600";
    area=800,600;
    maximum=yes;
    format=jpeg;
