Der Parameter form

Der Parameter form kann die Werte

  • tiff
  • jpeg
  • sdif
  • gif
annehmen. Er gibt das Datenformat der Zieldatei(en) an. Das jeweilige Datenformat der Ausgangsdateien wird durch das System erkannt. Die Werte entsprechen den üblichen Endungen der gängigen Bildformate.

Der Befehl

apply image=raw;
    target="res800x600";
    area=800,600;
    maximum=yes;
    format=jpeg;

würde also die Ausgabe der Bilddaten im Zielformat JPEG anordnen.
