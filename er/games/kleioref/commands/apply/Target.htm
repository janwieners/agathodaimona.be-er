Der Parameter target

Der Parameter target ewartet als Parameterwert eine Zeichenkette, die in Hochkommata eingeschlossen sein muß. Der Wert von Target gibt den Zielpfad an, in dem die durch apply errechneten Bilddateien abgelegt werden.

Die im Zielpfad angegebenen Verzeichnisse müssen vor Ausführung des apply-Befehls mit den Befehlen des jeweiligen Betriebssystems angelegt werden.

apply image=raw;
    target="res800x600";
    area=800,600;
    maximum=yes;
    format=jpeg;

Hier werden als Bilder mit der Auflösung 800 x 600 (vgl. Parameter area) in einem Verzeichnis abgelegt, das den Namen "res800x600" trägt.
