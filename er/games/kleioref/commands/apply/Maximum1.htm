Der Parameter maximum

Der Parameter maximum erwartet die Parameterwerte yes oder no. Die letztgenannte Einstellung ist der Default-Wert des Systems. Der Parameterwert yes weist das System an, die Größe des jeweiligen Zielbildes opimal in die Größe einzupassen, das durch den area-Parameter spezifiziert wurde.

Wird der maximum-Parameter weggelassen oder der Wert no spezifiert, so wird das Zielbild (unabhängig von allen Angaben im area-Parameter) jeweils in der Standardgröße von 1014 x 1250 angelegt.
