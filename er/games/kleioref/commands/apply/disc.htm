Der Befehl apply

Das Kommado apply ermöglicht die Bearbeitung von Daten des Types image. Im wesentlichen dienst es zur Umrechnung von Bildgrößen und zur Wandlung von Bildformaten.

Ein typischer apply-Befehl hat die folgende Form:

apply image=raw;
    target="res800x600";
    area=800,600;maximum=yes;
    format=jpeg;
    step=yes;
    target="res940x550";
    area=940,550;
    maximum=yes;
    format=jpeg;
