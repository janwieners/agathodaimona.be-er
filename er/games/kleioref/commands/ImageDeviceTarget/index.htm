Der Parameter Target

Der Parameter Target erwartet als Paramterwert eine Konstante, die in Hochkommata angegeben werden muﬂ. Der Inthalt von "target" entspricht dem Verzeichnis oder dem Laufwerk, in dem das im name-Parameter angegebene Verzeichnis vorliegt.

Es kann hier neben einem Laufwerk auch eine Maschine wie z.B. eine Jukebox anegeben werden, die als Laufwerk gemountet ist. Die Angabe hängt vom Betriebssystem und der Konfiguration des jeweiligen File-Systems ab. Eine typische Anwendung hat die folgende Form:

device name=scans;target="."
