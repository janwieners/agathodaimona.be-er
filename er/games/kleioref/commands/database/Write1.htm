Der Parameter write:


Der Parameter write weist das System an, bei der Anlage der Datenbank Grundinformationen über deren Struktur auszugeben. Die Funktion von write wird durch vier Parameterwerte gesteuert, die jeweils auf den ersten Buchstaben abgekürzt werden dürfen. Der Parameter kann wiederholt angegeben werden.

  • Der Parameterwert names weist das System an, Bezeichnung und Datentyp der jeweiligen Elemente einschließlich ihrer Kommentar- und Originaltextfelder explizit anzugeben. Zudem werden die logischen Objekte benannt, die auf den Datentyp des Elements Einfluß haben.
  • Der Parameterwert  parts ordnet die  Wiedergabe der Namen der Informationsgruppen innerhalb jeder Datenbank an. Zusätzlich werden die Eigenschaften der Gruppe und ihre Stellung innerhalb der Datenbank angegeben.
  • Der Wert structure weist das System an, daß die hierarchische Abhängigkeit der Gruppen (mit Angabe der darin enthaltenen Elemente) in einer Übersichtsdarstellung ausgegeben wird.
  • Der Parameter generic, der sich auf generische Datenbankinformationen bezeiht, ist gegenwärtig außer Funktion.
