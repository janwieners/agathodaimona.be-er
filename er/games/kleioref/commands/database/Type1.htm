Der Parameter type:


Der Parameter type regelt die Dauerhaftigkeit bzw. Nichtdauerhaftigkeit einer Datenbank. Durch Angabe des Parameterwerts permanent (zugleich standardmäßige Voreinstellung) wird das System angewiesen, eine Datenbank für dauerhafte Nutzung anzulegen. Der Parameterwert temporary legt dagegen fest, daß die Datenbank nach Abarbeitung des Gesamtjobs wieder gelöscht wird. Dies bedeutet, daß die Systemdateien vom Rechner entfernt werden. Beide Parameterwerte können auf den ersten Buchstaben abgekürzt werden.
