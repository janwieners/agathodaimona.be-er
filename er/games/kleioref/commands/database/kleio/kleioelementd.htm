Die Direktive Element


Die Direktive element dient dazu, einem Element eine bestimmte Eigenschaft zuzuweisen, die dann üblicherweise in der gesamten Umwelt für alle Elemente desselbe Namens gilt. Insbeondere dient element dazu, den Datentyp (text [Standard], date, number, category, relation, location oder image) eines Elements oder seiner Aspekte (Kommentar  und Originaltext) festzulegen. Weiterhin kann z.B. durch den Parameter connect festgelegt werden, wie der Inhalt verschiedener gleichnamiger Tags behandelt werden soll.

Einstweilen werden  hier nur die mit "*" gekennzeichneten Direktiven erläutert, die in der alten Systembeschreibung nicht enthalten sind.

Bis zur endgültigen Fertigstellung der WWW-Dokumentation wird hinsichtlich der übrigen Parameter  auf die bisherige Systemdokumentation , besonders Abschnitt 5 verwiesen.
