Der Paramter usage:

Der Parameter usage regelt das Verhalten der transparent-Direktive. Er kann z.Zt. zwei Werte annehmen:
  • delete
  • insert (=Standardverhalten)
Der Parameter insert bewirkt, daß ein transparentes Symbol in die Umgebung der Datenbank eingeführt wird. Dies ist der Standard des Systems. Das heißt, die im name-Parameter benannten Symbole werden entsprechend den Regeln des type-Parameters behandelt.

Dagegen entfernt der Parameter delete ein transparentes Symbol aus der Umgebung. Dies kann sich als notwendig erweisen, wenn mehrere Datenbanken nacheinander erstellt werden, in denen dieselben Symbole verwendet werden, die aber jeweils unterschiedlich behandelt  werden sollen. In diesem Falle kann z.B. die "Transparenz" eines Symbols für bestimmte Datenbanken aufgehoben  werden. Wenn der name-Parameter nicht spezifiziert ist, werden alle transparenten Symbole gelöscht.

transparent usage=delete
hebt also in der Gesamtumwelt als Ignorierungsanweisungen für bestimmte Tags auf.
