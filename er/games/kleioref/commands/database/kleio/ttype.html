Der Parameter type:

Der Parameter type erwartet einen der folgenden Parameterwerte:


  • ignore
  • tag (= Standardverhalten des Systems)
  • attribute
  • data
  • element

Der Wert ignore weist das System an, einen Tag, den dazugehörigen Inhalt sowie ggf. Attribute zu ignorieren. Das bedeutet mit anderen Worten, daß die gesamte Zeichenkette vom Start- bis zum Endtag nicht in die Datenbank übernommen wird.

Der Wert tag weist das System an, nur den Tag, nicht aber dessen Inhalt und Attribut zu ignorieren.

Ein typisches Anwendungsbeispiel für transparent könnte in folgendem Beispiel bestehen. Folgender Auszug aus einer XML-Datei soll in eine Datenbank eingelesen werden:

<text>Das ist ein Text,<kursiv> in dem bestimmte Teile durch kursive Schrift</kursiv> hervorgehoben sind.</text>

Der Gesamttext soll als ein Element in der Datenbank aufscheinen, das Tagging für die kursive Schrift (<kursiv></kursiv>) aber ignoriert werden. Die Direktive 

tranparent name=kursiv;type=tag

löst das Problem, da sie Auszeichnung für kursiv unterdrückt, den Inhalt aber als Teil des übergeordneten Tags "text" in die Datenbank übernimmt.


Der Wert attribute weist das System an, ggf. vorliegende Attribute zu bestimmten Tags zu ignorieren. In diesem Fall bezieht sich der name-Parameter also auf das betreffende Attribut.
transparent name=wert;type=attribute

heißt also, daß in allen Tags ein ggf. vorkommendes Attribut mit dem Namen "wert" ingnoriert wird. Dieses Verhalten besteht unanbhängig von der Tatsache, daß ggf. auch ein Tag mit dem Namen "wert" ignoriert werden kann. In diesem Falle bestehen zwei unabhängige transparent-Direktiven

transparent name=wert;type=attribute
transparent name=wert;type=tag

Die erste Direktive ordnet an, daß in allen Tags ein ggf. vorkommendes Attribut "wert" ignoriert wird. Die zweite Direktive legt dagegen fest, daß ein Tag des Namens "wert" nicht bearbeitet wird.

Der Wert data weist das System an, den Tag selbst als Teil der Daten zu interpretieren und sowohl den Tag als auch dessen Inhalt in die Datenbank zu übertragen.

Der Wert element ignoriert sowohl den tag als auch ggf. ein Attribut, übernimmt aber den Inhalt in die Datenbank.
