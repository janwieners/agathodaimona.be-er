Der Parameter association:

Der Parameter association kann die Parameterwerte local oder version annehmen. Ersteres ist Systemstandard. Der Parameter local bewirkt, daß eine bestimmte Eigenschaft sich nur auf die jeweilige Gruppe bezieht, in der die Eigenschaft als Tag oder Attribut aufscheint. Dagegen bewirkt version, daß die betreffende Eigenschaft auf alle enthaltenen Gruppen vererbt wird.
