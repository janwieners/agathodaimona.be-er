Der Parameter identification:

Der Parameter erwartet die Parameterwerte yes oder no. Ersteres ist der Standardwert.

Der Parameter steuert die Identifikation der einzelnen Dokumente in der Datenbank. Standardmäßig (= Parameterwert yes) werden diese dadurch identifiziert, daß Kleio jeweils das erste Element der dokumentbildenden Gruppe als Indentifikation heranzieht.

Wenn dieses nicht sinnvoll erscheint (z.B. weil das erste Element nicht eindeutig ist), so kann durch den Parameterwert no das System angewiesen werden, eine Standardidentifikation anzulegen, die aus dem Gruppenamen, einem Bindestrich und der Ordnungsnummer der Gruppe innerhalb der Datenbank besteht.
