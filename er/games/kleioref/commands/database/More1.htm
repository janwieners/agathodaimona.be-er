Der Parameter more:


Der Parameter more regelt die Unterscheidung von Identifikatoren in der Datenbank. Der Parameterwert no, der das Standardverhalten darstellt, setzt voraus, daß jeder Identifikator genau nur einmal in der Datenbank vorkommt. Dagegen erlaubt der Wert yes, daß identische Daten als Identifikationen herangezogen werden können. In diesem Fall ist das System für die Unterscheidung der Identifikatoren verantwortlich. Beide Parameterwerte dürfen auf den ersten Buchstaben abgekürzt werden.
