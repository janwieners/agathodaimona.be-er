Der Parameter name:


Der Parameter name erwartet als Parameterwert den Namen einer Datenbank, der prinzipiell frei wählbar ist.  Eine Unterscheidung zwischen Groß- und Kleinbuchstaben findet nicht statt. Die Datenbank kann später unter dem jeweils gewählten Namen angesprochen werden.

Die Dateien, in denen die Datenbank physikalisch verwaltet wird, tragen  den Namen der Datenbank, der in diesem Falle durch entsprechende Endungen erweitert wird.  Damit unterliegt die Länge des Datenbanknamens prinzipiell eventuellen Restriktionen des jeweiligen Betriebssystems.
