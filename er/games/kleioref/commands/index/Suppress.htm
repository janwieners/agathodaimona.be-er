Der Parameter suppress:

Suppress erwartet als Parameterwert eine durch Kommata getrennte Liste von Namen für logische Objekte der Klasse conversion. Der Parameter kann jeweils auf einzelne Sortierspalten angewendet werden und bewirkt, daß die aufgeführten Objekte jeweils vor der Sortierung nacheinander auf den Inhalt der Spalte angewendet werden.
