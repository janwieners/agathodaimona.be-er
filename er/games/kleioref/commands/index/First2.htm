Der Parameter first:


Standardmäßig  generiert der Index-Befehl eine aufsteigende Sortierfolge. Die Stellung der einzelnen Zeichen (Buchstaben, Ziffern, Sonderzeichen) innerhalb der Sortierordnung wird durch ein logisches Objekt der Klasse order bestimmt, das den Namen  order trägt.

Durch den Befehl

description name=order;type=order

kann man sich die standardmäßig eingestellte Sortierordung anzeigen lassen.  Ebenso ist es möglich, durch entsprechende Umdefinition des logischen Objekts order oder durch Neuanlage einer anderen Sortierordnung mittels des item -Befehls eine andere als die voreingestellte Sortierordnung anzulegen (vgl. hierzu auch den Parameter order ). Grundsätzlich aber gilt aber auch dann, daß alle Daten der gewählten Sortierordnung entsprechend in aufsteigender Folge ausgegeben werden.

 Durch Angabe des Parameters first mit dem parameterwert limit (implizite Standardvorgabe: start) kann man aber eine absteigende Sortierfolge explizit einstellen.

Der Befehl

index part=:name;first=limit

erzeugt damit folgendes Register:

ZWINK DOK051064
   
ZWEIGLEN DOK977124
   
ZWEIGLE DOK999390
   
  ....


ACKER DOK001199

ABT DOK872101

ABELE DOK050295
