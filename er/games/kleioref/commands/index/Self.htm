Der Parameter self:


Der Parameter self  erwartet als Parameterwert eine frei wählbare Zeichenkette, die durch einfache oder doppelte Hochkommata begrenzt werden muß. Der Parameter bewirkt, daß im Falle fehlender Angaben der Inhalt der Zeichenkette als Default-Wert in das Register  eingefügt wird:

Gegeben sei ein Register aus Personennamen und Herkunftsort (:hko) der jeweiligen Person. Der Herkunftsort liegt nicht immer vor. Dies kann man durch den Parameter self im Register entsprechend  deutlich machen:

index part=:name;
         part=:hko;self="keine Angabe"


Das Resultat macht die Funktion des Parameters deutlich:

ABT ESSL DOK872101
   
ABT JUNGINGEN DOK000085
   
ABT TUEBINGEN DOK870075
   
ACKER keine Angabe DOK001199
   
ACKERL keine Angabe DOK000147
   
ACKERL keine Angabe DOK000901
   
ACKERMANN VILLINGEN DOK872671

ADELMANN VON keine Angabe DOK030015

AFFENARZ keine Angabe DOK010003

Die Zeichenkette bei self hat standardmäßig keinen Einfluß auf die Sortierung. Dies kann durch den Parameter visibility umgestellt werden.
