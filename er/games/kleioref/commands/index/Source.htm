Der Parameter source:


Der Parameter source dient der Differenzierung innerhalb von Aufgaben mit mehreren Index-Kommandos. Der Parameter lines weist bei Angabe des Parameterwerts yes das System an, bei der Reihenfolge der Datenausgabe ggf. zu unterscheiden, ob eine Information durch den ersten, zweiten dritten usw. Index-Befehl erzeugt wurde. Durch Angabe des Parameters source, der als Parameterwert eine ganze Zahl erwartet, können den einzelnen Index-Befehlen explizit Nummern zugewiesen werden, die dann nach den Regeln  des lines -Parameters als Sortierkriterium gelten können. Dabei können mehrere Index-Befehle dieselbe Nummer erhalten.

Ein Beispiel wäre folgende Aufgabe:


index part=part[kar,1]:beruf; lines =yes;source=1
index part=part[kar,2]:beruf;source=2
index part=pdat[kar,3]:beruf;source=2
index part=pdat[kar,4]:beruf;source=2

Hier werden alle Angaben aus der ersten Zeile als "erster Index-Befehl" betrachtet, während die übrigen drei Befehle hinsichtlich der Sortierung als "zweiter Index-Befehl" gelten.
