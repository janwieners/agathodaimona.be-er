Der Parameter package:

Der Parameter package erlaubt, den Inhalt einzelner Sortierspalten mittels festgelegter Zeichenfolgen zu umschließen, also zu verpacken. Der hauptsächliche Einsatzzweck des Parameters ist die Aufbereitung der Datenausgabe nach HTML-Konventionen, um ein Listing auf diese Weise "über das Web" schicken und in einem Browser darstellen zu können.

Als Wert erwartet package den Namen eines logischen Objekts (i.e. einer Definition) der Klasse package, das festlegt, welche Zeichen (i.e. Tags) vor und nach der jeweiligen Information ausgegeben werden sollen.

Als Beispiel dient die folgende Vereinbarung mit dem Namen "pack":

item name=pack;usage=package;overwrite=yes
fields start="<b>";
       limit="</b><br>"
exit name=pack

Sie legt fest, daß jeweils vor Anfang und Ende der Information die Tagfolge "<b>" bzw. "</b><br>" ausgegeben werden sollen.

Nimmt man in einem Index-Befehl darauf Bezug, so erhält die gesamte Aufgabe die folgende Form:

query name=basdat;part=per
index part=:name;package=pack;type=list
stop 


Das Ergebnis ist ein Listing, das in einem Browser als fettgedruckte Namenliste dargestellt würde:

<b>ABELE</b><br>
     
<b>ABELE</b><br>
     
<b>ABT</b><br>
     
<b>ABT</b><br>
     
<b>ABT</b><br>

         .....

<b>ZWEIGLEN</b><br>

<b>ZWEIGLEN</b><br>

<b>ZWINK</b><br>


In einem Browser sieht dieselbe Liste folgendermaßen aus:


ABELE
ABT
ABT
ABT
......
ZWEIGLEN
ZWEIGLEN
ZWINK



Dieser Effekt ließe sich grundsätzlich auch durch Einsatz der Funktion package erzielen.


Dennoch würden die folgenden beiden Befehle in einzelnen Fällen ein unterschiedliches Ergebnis liefern:

Befehl 1:
index part=:form[" "];part=:package[:name,pack];type=list

Befehl 2:
index part=:form[" "];part=:name;package=pack;type=list

Im Falle von Befehl 1 würde package in denjenigen Fällen, in denen :name nicht vorliegt nicht ausgeführt. Dagegen werden die Verpackungsanweisungen, die durch den Parameter package (wie in Beispiel 2) angeordnet werden, auch dann ausgeführt, wenn das zu verpackende Element nicht vorliegt. Dies ist insbesondere Dann von Bedeutung, wenn der Inhalt des Package nicht nur Formatierungsanweisungen, sondern auch Textbestandteile enthält, die bei der Lösung in Befehl 1 nicht ausgegeben würden.
