Der Parameter text:

Der Parameter  text erwartet die Parameterwerte yes oder no. Letzeres ist Standard. Die Angabe "text=yes" bewirkt, daß in der jeweiligen Sortierspalte aufeinanderfolgende Leerstellen standardmäßig durch eine einzige Leerstelle ersetzt werden.

Der Parameter kann besonders sinnvoll sein, wenn durch Anwendung mehrerer Objekt der Klasse conversion (etwa durch den parameter suppress ) verschiedene Zeichenkette jeweils durch Leerstellen ersetzt werden. Daraus ergibt sich eine Folge von mehreren Leerstellen, die durch den Parameter text durch ein Leerstelle ersetzt werden kann.
