Der Parameter signs:

Der Parameter signs dient der Einstellung der einzelnen Spaltenbreiten.  Er bezieht sich jeweils nur auf eine, durch einen part-Parameter eröffnete  Registerspalte. Als Wert erwartet signs eine beliebige Zahl, die wiederum  die Anzahl der Zeichen repräsentiert, die für die Spalte reserviert  werden. Die Eingabe der Zahl erfolgt ohne Anführungszeichen.

Wenn ein Spalteneintrag mehr Zeichen umfaßt als im signs-Parameter  angegeben, so wird der Eintrag abgeschnitten.

Ein Anwendungsbeispiel für den signs-Parameter ist folgende Aufgabe:


index part=:name;signs=10;
          part=:coll[:vorname];signs=20


  Das Resultat dieses Index-Befehls ist eine dreispaltiges Register. Dabei  sind in der ersten Spalte für die Elementarinformation :name zehn Stellen  reserviert, für die Gesamtheit aller Vornamen (:collection[:vorname])  ist eine Spaltenbreite von 20 Zeichen vorgesehen. Die anschließenden  Dokumentenidentifikation wird standardmäßig ausgegeben: Zeichenketten, die die voreingestellte Breite überschreiten, werden abgeschnitten:
