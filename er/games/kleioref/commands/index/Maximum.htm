Der Parameter maximum

Der Parameter maximum erwartet als Parameterwert eine ganze Zahl, die ohne Hochkommata angegeben wird. Im Standardverhalten des Indexbefehls ( type =list oder type=index)  bewirkt der Parameter, daß wiederholte Informationen identischen Inhalts nur so oft ausgegeben werden, wie durch den Parameterwert eingestellt. Das heißt, die Zahl der identischen Ausgabe- bzw. Registerzeilen wird auf den Parameterwert reduziert

Nehmen wir an, in einer Datenbank liegen Berufsangaben vor, so wird uns u.U. nicht in jedem Falle die Häufigkeit der einzelnen Berufe interessieren, sondern die Tatsache des Vorkommens eines Berufsbelegs an sich. Durch Angabe des Parameters maximum mit dem Parameterwert 1 erreicht man, daß jede Berufsvariante genau einmal ausgegeben wird:

index part=:beruf;maximum=1
