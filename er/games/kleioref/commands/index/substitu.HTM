Der Parameter substitution


Der Parameter substitution erlaubt die Ersetzung wiederkehrender Spalteneinträge durch eine frei wählbare Zeichenkette, die als Parameterwert zwischen Hochkommata eingeschlossen angegeben werden muß. Im Falle längerer Register kann dieser Parameter die Lesbarkeit der Listings erhöhen, da der jeweilige Wechsel eines Begriffs deutlich hervorgehoben werden kann.

Die Funktion des Parameters verdeutlicht folgendes Beispiel:

index part=:beruf;substitution="-----";
        part=:von-bis


Hier werden also wiederholte Berufsangaben so lange durch die Zeichenkette '-----' ersetzt, bis ein neuer Begriff erscheint. Der Wechsel ist damit deutlich markiert:

ACCOUCHEUR 1825.00.00 DOK000629

ACTUAR 1842.00.00 DOK810012
   
ACTUAR,GERICHT 1841.07.01 DOK000516
   
----- 1845.07.01 DOK050900
   
----- 1846.07.01 DOK000509
   
----- 1847.00.00 DOK001338
   
----- 1848.05.00 DOK020447
   
----- 1849.07.01 DOK050834
   
----- 1850.07.01 DOK030642
   
----- 1850.07.01 DOK050480
   
ACTUAR,OBERAMT 1840.00.00 DOK834784


Eine Erweiterung des Systemverhaltens kann erreicht werden, wenn zusätzlich der Parameter cumulate mit dem Parameterwert yes angegeben wird. In diesem Falle bezieht sich die Ersetzungsanweiseung durch substitution nicht nur auf eine logische Spalte, sondern auf alle Spalten beginnend mit jener, für die substitutuion angegeben wurde, bis hin zu jener Spalte bei der cumulate eingetragen wird.
