Der Parameter catalogue

Der Parameter catalogue steuert die Ausgabe von Meldungen bei der Abarbeitung von Katalogen. Standardmäßig werden Fehlermedungen in vollem Unfang ausgegeben. Die Eingabe von

options catalogue=mute

unterdrückt die Ausgabe von Meldungen. Dies kann z.B. bei Webservern sinnvoll sein, wenn fehelden oder illegale Suchbegriffe längere Fehlermeldungen produzieren, die im Kontext generierter HTML-Seiten Outputs erzeugen, die den unbedarften User verwirren können. Das o.g. Options-Kommando verhindert dies.
