Das Kommando execute

Das Kommando execute dient dazu, eine katalogisierte Prozedur auszuführen. Dies kann sowohl lokal als auch in Web-Anwendungen innerhalb eines CGI-Directorys geschehen.

Man habe folgende einfache katalogisierte Prozedur:

item name=job1;usage=procedure;overwrite=yes
cata source="query name=piranesi\nwrite"
exit name=job1

Es wird hier also eine Aufgabe namens "job1" katalogisert, die den Inhalt der Datenbank "piranesi" mittels eines "write-Kommandos" wiedergibt. durch den folgenden execute-Befehl kann dieses Prozedur aufgerufen werden:

exec name=job1

Als Beispiel für den Aufruf im Rahmen einer CGI-Anwendung kann der folgende URL  dienen. Der aufmersame Leser wird bemerken, daß darin ein execute-Kommando (abgekürzt als "exec") enthalten ist:

http://gilgamesch2.hki.uni-koeln.de/prometheus-cgi/kleioc/1010KlProm/exec/zoom/%223475%22/__kleioid/%22gast4%5fgast4%5fh%2ekoeln%22/__kleiopass/%222%2e2%2e20000%22
