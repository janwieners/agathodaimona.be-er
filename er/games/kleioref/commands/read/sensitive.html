Der Parameter sensitive

Der Parameter sensitive gehorcht auf die Parameterwerte yes (Standard) und no. Der Wert no ordnet an, daß bei der Verarbeitung von Daten, die in einer Markupsprache (üblichweise XML oder SGML) vorliegen, im Hinblick auf die Markups keine Unterscheidnung zwischen Groß- und Kleinschreibung getroffen wird.

Der Tag <name> enstpricht damit den Varianten <Name> oder <NAME>.



Achtung:

Unter Kleio ist der Tag  <KlEnt/> als Voreinstellung zur Kennzeichnung von Mehrfacheinträgen vorgesehen. Die Verwendung des vordefinierten Tags  <KlEnt/> setzte gegenwärtig zwingend die Einstellung sensitive=yes voraus.
