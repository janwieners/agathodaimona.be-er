Der Parameter name:


Der Parameter name ist obligatorischer Betstandteil jedes read-Befehls. Er bezieht sich auf den Namen einer Datenbankvereinbarung, die zuvor durch den database-Befehl in die lokale Umwelt eingeführt worden sein muß. 

Damit werden die Rohdaten entsprechend der zugehörigen Strukturvereinbarung abgearbeitet.


Der Befehl

read name=fre


weist das System also an, die anchfolgenden Rohdaten entsprechend den vereinbarungen einzulesen, die im database-Befehl für die Datenbank "fre" getroffen wurden.
