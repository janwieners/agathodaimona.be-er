Der Parameter line:

Der Parameter line erwartet als Parameterwert eine ganze Zahl zwischen Null unsd vier. Er entspricht in seiner Wirkung dem Äquvalent im options -Befehl: In den verarbeiteten Daten werden entsprechend dem Parameterwert leere Zwischenzeilen eingeschoben. Der Parameter line ist sinnvoll in Verbindung mit repeat=yes .

Line übernimmt standardmäßig eine Voreinstellung durch den Options-Befehl.
