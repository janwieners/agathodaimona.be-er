Der Parameter cumulate


Der Parameter cumulate bezeihts ich auf Daten im "klassischen" Kleio-Eingabeformat. Er erwartet ein einzelnes Zeichen, das als Ersetzungzeichen für indentischen Inhalt bei gleichnamigen Elementen gilt.


Beispiel:

Familie$Nummer123
Vater$name=Huber/vorname=Peter
Mutter$"/Maria


Wenn dieser Datensatz mit dem Parameter cumulate="  eingelesen wird, wiederholt Kleio in der Informationsgruppe Mutter$ die Angabe des Inhalts des Elements name in der Gruppe Vater$
