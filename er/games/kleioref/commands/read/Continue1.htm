Der Parameter continue:


Der Parameter continue kann die Werte clio oder none annehmen.Ersteres ist der Standardwert.

Der Wert clio weist das System an, Zeilenumbrüche in den Eingabedaten zu ignorieren und jeweils das erste Zeichen der Folgezeile unmittelbar an das letzte Zeichen der vorherigen Zeile anzuschließen. Eventuelle Leerzeichen am Ende der vorherigen Zeile werden ignoriert. Der Wert none legt fest, daß Folgezeilen jeweils durch eine Leerstelle verbunden werden.

Der Parameter continue ist nur für Daten von Bedeutung, die im "klassischen" Eingabeformat vorliegen. Er hat keinen Einfluß auf Daten im Markup-Format. In letzterem Format werden gegenwärtig Zeilenumbrüche generell als Leerstellen gewertet.
