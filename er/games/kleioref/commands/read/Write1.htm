Der Parameter write:


Der Parameter write kann mehrmals angegeben werden und erwartet jeweils einen der folgenden  Parameterwerte

  • names
  • structure
  • parts

Der Parameterwert names bewirkt, daß nach dem Einlesen der Daten eine Übersicht über die Namen und Eigenschaft der definierten Elemente einer Datenbasis ausgegeben wird.

Der Wert structure weist das System an, nach dem Einlesevorgang eine schematische Übersicht über die Struktur der Datenbank (Gruppen, Elemente, Hierarchie) zu erstellen.

Der Wert parts veranlaßt das System, nach dem Einlesevorgang eine Auflistung über die definierten Gruppen und deren Eigenschaften auszugeben.
