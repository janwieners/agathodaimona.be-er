Der Parameter second

Der Parameter second entspricht in seiner Handhabung dem Parameter first.

Während first die erste Wiederholung einer Information und deren entsprechende Auszeichung bewirkt, kann mit second eine zweite Wiederholung einer Information mit zugehöriger Auszeichnung angeordnet werden. Ein entsprechendes Package hat demnach diese Form:

item name=paket;usage=package;overwrite=yes
   field start="Anfang";
   first="erste Wiederholung";
   second="zweite Wiederholung";
   limit="Ende"
exit name=paket
